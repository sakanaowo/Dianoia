# 04 - Notes

Đây là trái tim của vault.

Thư mục này chứa các note đã được xử lý, viết lại và liên kết. Không phải mọi ghi chú đều cần đưa vào đây. Chỉ những ý tưởng có giá trị lâu dài hoặc cần được nối với hệ thống kiến thức mới nên nằm ở đây.

## Các loại note

```text
Atomic
Concepts
Arguments
Comparisons
Questions
People
```

## Atomic note

Atomic note là note chỉ chứa một ý chính.

Ví dụ tên tốt:

```text
Temperature làm phân phối token sắc hoặc phẳng hơn.md
Một niềm tin đúng chưa chắc đã là tri thức.md
Tự do tiêu cực là tự do khỏi sự can thiệp.md
CNN học đặc trưng cục bộ bằng convolution kernel.md
```

Tên chưa tốt:

```text
Temperature.md
Knowledge.md
Freedom.md
CNN.md
```

## Concept note

Dùng để giải thích một khái niệm.

Ví dụ:

```text
Softmax.md
Entropy.md
Social Construct.md
Free Will.md
Capitalism.md
```

Concept note nên có định nghĩa, ví dụ, hiểu nhầm thường gặp và liên kết.

## Argument note

Dùng cho triết học, xã hội, chính trị, kinh tế, đạo đức.

Một argument note nên có:

```md
# {{title}}

## Luận điểm

## Lập luận ủng hộ

## Lập luận phản đối

## Giả định ngầm

## Điểm yếu

## Ví dụ thực tế

## Liên kết
```

## Comparison note

Dùng để so sánh hai hoặc nhiều khái niệm.

Ví dụ:

```text
Supervised learning vs Unsupervised learning.md
Existentialism vs Nihilism.md
Capitalism vs Socialism.md
Negative liberty vs Positive liberty.md
```

## Question note

Dùng cho các câu hỏi chưa có câu trả lời chắc chắn.

Ví dụ:

```text
Máy có thể suy nghĩ không.md
Con người có tự do ý chí không.md
Làm sao biết điều gì là thật.md
```

## Luật cho Notes

- Mỗi note nên nói một ý rõ ràng.
- Mỗi note quan trọng nên có link đến note khác.
- Không copy nguyên văn quá nhiều từ nguồn.
- Viết bằng lời của mình.
- Nếu một note quá dài, tách thành nhiều atomic note nhỏ hơn.
