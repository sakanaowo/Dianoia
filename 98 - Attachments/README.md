# 98 - Attachments

Attachments là nơi lưu file đính kèm của vault.

Ví dụ:

```text
Images
PDFs
Screenshots
Diagrams
Audio
Exports
```

## Nội dung phù hợp

- Ảnh chụp màn hình.
- Hình minh họa.
- PDF.
- Biểu đồ.
- File xuất từ công cụ khác.
- Hình dùng trong báo cáo hoặc note.

## Cách tổ chức đề xuất

```text
98 - Attachments/
├── Images/
├── PDFs/
├── Screenshots/
├── Diagrams/
└── Exports/
```

## Luật cho Attachments

- Không ghi note trực tiếp trong thư mục này.
- Không để file rác tích tụ quá nhiều.
- File quan trọng nên được link từ note liên quan.
- Nếu một PDF hoặc tài liệu có nhiều ghi chú, tạo Source note trong `05 - Sources` rồi link tới file đính kèm.
