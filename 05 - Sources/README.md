# 05 - Sources

Sources là nơi lưu ghi chú từ nguồn bên ngoài.

Nguồn có thể là:

```text
Books
Papers
Articles
Videos
Podcasts
Courses
Quotes
Documentation
```

## Source note dùng để làm gì?

Source note dùng để ghi lại thông tin từ một nguồn cụ thể.

Ví dụ:

```text
Books/Nietzsche - Beyond Good and Evil.md
Papers/Attention Is All You Need.md
Videos/CS231n Lecture 1.md
Documentation/PyTorch ImageFolder.md
```

## Source note không phải là permanent note

Một lỗi phổ biến là ghi chép rất nhiều từ sách, video, paper rồi để nguyên ở đó.

Source note chỉ là bước trung gian.

Kiến thức quan trọng nên được tách sang `04 - Notes`.

Ví dụ:

```text
Source note:
05 - Sources/Books/Nietzsche - Beyond Good and Evil.md

Atomic notes:
04 - Notes/Atomic/Nietzsche phê phán đạo đức bầy đàn.md
04 - Notes/Atomic/Sự thật có thể bị chi phối bởi nhu cầu sống của con người.md
```

## Template source note

```md
# {{title}}

## Thông tin nguồn

- Tác giả:
- Loại nguồn:
- Link:
- Năm:
- Chủ đề:

## Tóm tắt ngắn

## Ý chính

## Câu hỏi của mình

## Trích dẫn đáng chú ý

## Có thể tách thành note

- [[...]]
- [[...]]
```

## Luật cho Sources

- Ghi rõ nguồn nếu note đến từ sách, paper, video hoặc tài liệu.
- Không cố biến Source note thành bài viết hoàn chỉnh.
- Sau khi đọc xong, nên tách ra ít nhất vài atomic note nếu nguồn thật sự có giá trị.
