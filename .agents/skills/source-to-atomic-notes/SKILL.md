---
name: source-to-atomic-notes
description: Extract durable notes from books, papers, videos, articles, courses, documentation, or long source notes in the διάνοια vault.
---

When processing a source:

1. Identify source type and existing metadata.
2. Do not treat the source note as final knowledge.
3. Extract only ideas that are useful beyond the source.
4. Create atomic notes, concept notes, argument notes, or comparison notes as appropriate.
5. Link every extracted note back to the source.
6. Add extracted notes to relevant Domain or Theme MOCs when useful.
7. Preserve citations, page numbers, timestamps, or links when present.
8. If the source contains weak, biased, or unsupported claims, mark that clearly.

Output should include:

- Source processed.
- Notes created.
- MOCs updated.
- Claims that need verification.
