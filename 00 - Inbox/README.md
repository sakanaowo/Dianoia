# 00 - Inbox

Inbox là nơi ghi nhanh mọi thứ chưa kịp xử lý.

Đây là khu vực tạm thời, không phải nơi lưu trữ lâu dài. Mọi ý tưởng, câu hỏi, đoạn trích, lỗi code, suy nghĩ, link, video, bài giảng hoặc khái niệm mới đều có thể được ném vào đây trước.

## Khi nào dùng Inbox?

Dùng Inbox khi:

- Chưa biết note thuộc lĩnh vực nào.
- Chỉ muốn ghi nhanh để khỏi quên.
- Đang học hoặc đọc tài liệu và chưa muốn phân loại ngay.
- Có một câu hỏi cần quay lại sau.
- Gặp lỗi code hoặc vấn đề kỹ thuật cần xử lý.
- Có một ý tưởng xã hội, triết học hoặc cá nhân chưa đủ rõ.

## Nội dung phù hợp

Ví dụ:

```md
- Tại sao tính perplexity có </s> nhưng không tính <s>?
- Temperature trong LLM có liên quan gì đến entropy?
- Chủ nghĩa hiện sinh khác chủ nghĩa hư vô ở đâu?
- Vì sao xã hội cần luật pháp nếu con người có đạo đức?
- Lỗi CMake multiple definition trong CLion.
```

## Cách xử lý Inbox

Định kỳ chuyển nội dung trong Inbox sang nơi phù hợp:

```text
Ý tưởng đã hiểu rõ       → 04 - Notes
Nguồn tài liệu            → 05 - Sources
Câu hỏi lớn               → 03 - Themes hoặc 08 - Review
Việc đang làm             → 06 - Projects
Bài viết / tài liệu tạo ra → 07 - Outputs
Không còn cần nữa         → 07 - Archive hoặc xóa
```

## Luật cho Inbox

- Inbox được phép lộn xộn.
- Inbox không được bị bỏ mặc quá lâu.
- Nếu một ghi chú ở Inbox quá 1 tuần, cần quyết định: xử lý, lưu, chuyển, hoặc xóa.
