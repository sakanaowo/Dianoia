# 99 - Templates

Templates chứa các mẫu note dùng lại nhiều lần.

Mục tiêu của template là giảm ma sát khi tạo note mới, không phải ép mọi note vào một khuôn cứng.

## Template nên có

```text
Atomic Note.md
Concept Note.md
Argument Note.md
Comparison Note.md
Question Note.md
Source Note.md
Book Note.md
Paper Note.md
Project Note.md
Domain MOC.md
Theme MOC.md
Problem Note.md
Weekly Review.md
```

## Khi nào dùng template?

Dùng template khi:

- Tạo note mới có cấu trúc lặp lại.
- Ghi chú từ sách, paper, video, bài giảng.
- Viết atomic note.
- Phân tích một lập luận.
- Tạo project mới.
- Tạo MOC cho domain hoặc theme.

## Luật cho Templates

- Template nên ngắn.
- Không cần điền hết mọi mục trong template.
- Nếu một template làm bạn ngại viết note, template đó đang quá phức tạp.
- Template nên được sửa theo thói quen thực tế, không cần hoàn hảo từ đầu.
